# ExplainFX 

A minimal, simple and easy to use canvas tool created in JavaFX. Can be used to explain concepts during presentations or meetings. Inspired by https://excalidraw.com 

<img width="1279" height="743" alt="explainFX" src="https://github.com/user-attachments/assets/e68b5bf1-700f-41ef-8d57-587cee0db578" />

It's a very basic canvas app with only the most basic features. I created this for fun (I love JavaFX) but I will be slowly be adding more features. Developed with Java 21 and JavaFX 21. 

## Features 
1. Drawing 
2. Create Squares and Circles
3. Text
4. Vary stroke size/font size
5. Copy, paste, duplicate, lock, delete and move objects
6. Move camera via mouse

## Features to be added 
1. Save and load files
2. Edit or modify drawables after they have been drawn
3. more shapes

## To run on your machine 

1. For MacOS
   
Download the *.dmg installer* in release section

3. For Windows/Linux

*No distributions yet*, as JPackage requires the platform I am building for. (I only have access to macOS)

3. Within IntelliJ
- Clone the repository ``` https://github.com/gufranthakur/ExplainFX.git ```
- Download Java 21 (Eclipse Temurin JDK) from IntelliJ, if not installed
<img width="444" height="417" alt="image" src="https://github.com/user-attachments/assets/15359305-9741-48d7-8d08-1e92c15de575" />

- In the gradle tab, *ExplainFX > Tasks > Application > Run*
